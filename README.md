# Amazon Sales Analysis Dashboard

## 📊 Project Overview

This project presents an interactive Power BI dashboard built using Amazon sales data containing over **1 million records**. The dashboard provides insights into sales performance, revenue trends, product performance, customer behavior, and key business KPIs through dynamic visualizations.

## 📁 Dataset

* 1,000,000+ rows of sales data
* Customer, Product, and Order information
* Revenue and Sales metrics
* Multiple categories and regions

## 🛠️ Tools & Technologies

* Power BI
* DAX
* Power Query
* Excel
* Data Visualization

## 📈 Key Features

* Sales Trend Analysis
* Revenue Performance Tracking
* Product Category Analysis
* Customer Insights
* KPI Dashboard
* Interactive Filters & Slicers

## 🔍 Key Insights

* Identified top-performing products and categories.
* Analyzed monthly and yearly sales trends.
* Monitored business KPIs for performance evaluation.
* Compared sales performance across categories and regions.
* Generated actionable insights for data-driven decision-making.
